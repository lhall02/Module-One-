Artifact Description

Database Layer and API Integration

The selected database artifact is the MongoDB/Mongoose trip model and the Travlr API routes that create, retrieve, update, delete, and query trip records. The artifact originated from coursework using MongoDB with a Node/Express application.

The enhancement improves the database layer by adding stronger model rules, validating route input, returning clearer error responses, and connecting the Angular admin project to the trip API. This makes the project more representative of a complete web application that depends on reliable data persistence.

Enhancement

What Changed

Schema Validation

The Trip schema includes required fields, trimming, uppercase trip codes, unique trip codes, minimum price validation, and timestamps.

Defensive API Behavior

The API validates object IDs, required fields, prices, dates, and query ranges before database operations are completed.

Query Usefulness

Filtered queries, grouped resort summaries, date-range lookup, and sorted results make the data layer more valuable to the application.

Admin Integration

The Angular admin service and CRUD components connect to `/api/trips`, supporting add, edit, delete, and list workflows.

Reflection

Skills and Course Outcomes

The database enhancement reinforced the importance of thoughtful schema design and controlled data access. Data services are not just about storing information; they must support the application purpose safely, consistently, and effectively. I learned to think more carefully about invalid input, missing data, unique identifiers, and how errors should be communicated back to the caller.

This enhancement supports the course outcome related to security because it anticipates common data problems and reduces the chance of unsafe or malformed requests reaching the database. It also supports the outcomes for designing computing solutions and using well-founded tools because the work uses Mongoose, Express routing, RESTful conventions, and Angular services to deliver a more complete application.
