## Main Artifact

Travlr Getaways is a full-stack travel application using Express, Handlebars, MongoDB/Mongoose, and an Angular admin interface. The capstone enhancements improve customer workflow, algorithmic data handling, database validation, API behavior, and portfolio presentation.
